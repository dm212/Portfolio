# Yashwanth Portfolio

Static portfolio site for D M Yashwanth.

## Files

- `index.html` - main portfolio page
- `styles.css` - visual styling and responsive layout
- `script.js` - scroll reveal animation
- `assets/Yashwanth_Resume.pdf` - downloadable resume

## Preview locally

From this folder:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Easy hosting options

### GitHub Pages

1. Create a GitHub repository and upload these files.
2. In the repository settings, open `Pages`.
3. Set the source to the main branch root.
4. Your site will be published on a `github.io` URL.

### Netlify

1. Create a new site from a Git repository or drag and drop this folder.
2. No build command is needed.
3. Publish directory should be `/`.

### Vercel

1. Import the project.
2. Framework preset: `Other`.
3. No build command is needed.
4. Output directory can be left empty.
